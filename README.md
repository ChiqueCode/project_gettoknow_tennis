~ Tournament data
~ Match scores
~ Match statistics

====================== Step 2 ======================

Dump the data into SQL Database

====================== Step 3 ======================

Create a webpage using JavaScript, make it interactive

====================== Step 4 ======================

Load the data into a webpage

====================== Step 5 ======================

Using d3 visialize the data
